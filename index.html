Key Production Safeguards

  • Grounded Verification: The RAG agent is prohibited from answering if the matched score is below 85% confidence, replying with a standard fallback: *"I cannot confirm your eligibility for this scheme. Please consult the nearest Jan Seva Kendra."*
  • Code-Mixing Resolution: A preprocessing translation layer converts Hinglish and Tanglish inputs into standardized queries (e.g. *"Aadhaar card details link check"* matches the required documents field).
  • Bandwidth Mode: For basic 2G SMS/WhatsApp, the agent suppresses long explanatory texts, replying with standard numbered choices (e.g., *"Reply 1 for PM-KISAN, 2 for PMJAY"*).
